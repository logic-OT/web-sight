# Installing and Executing program

1. Clone repository
    ```
    git clone git@github.com:logic-ot/web-sight.git
    ```
2. Installing Depedencies
    ```
    pip install -r requirements.txt
    ```
3. Run flask
   ```
   python app.py
   ```
